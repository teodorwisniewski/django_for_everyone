Some dive at the begging

You must have at least one example of each of the following tags:

  • span,
  • p,
  • h1,
  • ul.
You must have three a tags and three li tags. You must have some bold text in the document and some italics text in the document but you must use the correct tags (i.e. neither b nor i are allowed>). There is a lot of flexibility within those parameters. Useful resources
